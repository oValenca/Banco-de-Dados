Projeto SisGESC - Sistema de Gestão Escolar

Este projeto implementa um sistema de banco de dados completo para gestão escolar, integrando os módulos Acadêmico, Financeiro e RH.

Como executar
1. Abra o MySQL Workbench.
2.Entre no seu servidor root.
3. Execute o script `sisgesc_atualizado(2).sql`.
4. O script realizará o reset do ambiente, criação das tabelas operacionais e analíticas, além da carga de dados idempotente.
